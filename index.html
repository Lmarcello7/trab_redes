Funcionalidades

Seja bem vindo, este é um jogo para adivinhar um numero gerado pelo computador, trabalhando com sockets.

Dificuldades:
  • Fácil: Tempo para responder de 30 segundos.
  • Médio: Tempo para responder de 15 segundos.
  • Difícil: Tempo para responder de 10 segundos.
Tecnologias utilizadas:
  • Jquery
  • Bootstrap 4
  • Html 5
  • Css 3
  • Php
